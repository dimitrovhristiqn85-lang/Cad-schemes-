# CAD Schematic Designs

This repository contains various CAD schematic designs developed for academic, school, and personal projects.

## Astable Multivibrator (Square Wave Generator)

This project features an astable multivibrator circuit designed to generate square-wave impulses. 

### Simulation Details
* **Analysis Type:** `TRAN` (Transient Analysis)
* **EDA Software:** KiCad
* **Status:** Fully functional simulation demonstrating expected transient behaviors and state switching under ideal initial conditions.
